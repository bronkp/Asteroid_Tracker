This project was made using Nasa's Near Earth Object Webservice and Nasa Image archive for background images. A list of Nasa's apis can be found here.

Code:

This website was made for fun to learn Express.js and practice CSS. The sizes of objects are altered with JavaScript based on the ratios of the objects size. You can get the raw data by going to /raw.

JSON Response Units:

Diameter Estimates: Feet

Speed: Miles per Hour

Distance: Miles
