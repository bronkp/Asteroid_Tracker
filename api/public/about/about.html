Data:

This project was made using Nasa's Near Earth Object Webservice and Nasa Image archive for background images. A list of Nasa's apis can be found at api.nasa.gov.

What is a potentially hazardous object?

Potentially Hazardous Asteroids or PHAs are defined by their ability to someday make a dangerous close approach to earth. The object just needs to get within a minimum distance of 4.6 million miles and be larger than around 500 feet in diameter. This casts a wide net that allows a high number of objects to be counted as hazardous, while much fewer are a real threat.

Source: cneos.jpl.nasa.gov
